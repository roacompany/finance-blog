# 토스 스타일 블로그 글쓰기 템플릿

> 토스의 5가지 코어밸류: Clear, Concise, Casual, Respectful, Empathetic을 기반으로 한 템플릿
> (참고: [토스의 8가지 라이팅 원칙](https://toss.tech/article/8-writing-principles-of-toss))

---

## 📊 토스 스타일 핵심 가이드

### 제목 작성법

**DO ✅**
```
"왜 예금보다 적금 이자가 적을까?"
"월세 vs 전세, 나한테 맞는 건?"
"2026년 1월 달라진 금융 혜택 총정리"
```

**DON'T ❌**
```
"예금과 적금의 금리 구조 비교 분석"
"월세 및 전세의 장단점에 대한 고찰"
"2026년 금융 제도 변경사항 안내"
```

**패턴:** 질문형 (90% 이상), "왜/어떻게/무엇이" 시작, 일상 언어, 호기심 유발

### 톤앤매너 변환표

| 기존 (격식체) | 토스 스타일 (친근체) |
|--------------|---------------------|
| ~입니다 | ~예요/이에요 |
| ~하시기 바랍니다 | ~해보세요/하시면 돼요 |
| 귀하 | (삭제) or 여러분 |
| ~에 대해 설명드리겠습니다 | ~을 알려드릴게요 |
| 상기와 같이 | 위에서 본 것처럼 |
| ~할 수 있습니다 | ~할 수 있어요 |

### 용어 설명 타이밍 (즉시성 원칙)

```
❌ 나쁜 예:
본문에서 COFIX를 10번 사용한 뒤 하단에 용어 설명

✅ 토스 방식:
COFIX를 첫 등장 시점에 바로 설명 (괄호나 Callout 사용)
"COFIX는 은행이 돈을 모으는 데 드는 평균 비용이에요. (Cost of Funds Index의 약자)"
```

### 콘텐츠 구조 패턴

**A. 라이프스타일형** — 일상 + 금융 연결, 흥미로운 도입부 → 자연스러운 연결 → 핵심 정보 → 생활 팁
**B. 금융 설명형** — 궁금증 유발 질문 → 즉시 용어 설명 → 비교 표 → 실생활 예시 → 3줄 요약
**C. 혜택/가이드형** — 새로 나온 것 → 단계별 설명 → 주의사항 → FAQ

### ✅ DO (이렇게 써요)

```markdown
❌ "기준금리는 한국은행 금융통화위원회에서 결정하는 정책금리입니다."
```

### 📏 문장 길이 가이드

- **한 문장**: 최대 1.5줄 (모바일 기준)
- **한 문단**: 2-3문장
- **리스트 항목**: 1줄로 끝내기

### 🎨 이모지 활용
