# SEO Rules (검색 최적화 규칙)

금융 블로그의 검색 노출을 최적화하기 위한 필수 규칙입니다.

---

## 1. 제목 구조 규칙

### H1 태그 (필수 1개)
- 페이지당 **H1 태그는 반드시 1개**만 사용
- 타겟 키워드를 H1에 포함
- 60자 이내 권장

### 제목 계층 구조
```
H1 - 메인 제목 (1개)
  H2 - 주요 섹션
    H3 - 하위 섹션
      H4 - 세부 항목
```

---

## 2. 필수 섹션 (배포 전 체크)

다음 섹션이 누락되면 **배포 불가**:

### TL;DR (핵심 요약)
- 글 상단에 3-5개의 핵심 포인트 요약
- 바쁜 독자를 위한 빠른 정보 제공

### FAQ (자주 묻는 질문)
- 최소 3개의 Q&A 포함
- 구조화된 데이터(Schema)로 검색 노출 향상
- 실제 검색 쿼리 기반 질문 작성

### 내부 링크 섹션
- 관련 글 최소 2-3개 링크
- 독자의 추가 탐색 유도
- 사이트 체류 시간 향상

---

## 3. 메타데이터 규칙

### Title (제목)
- 60자 이내
- 타겟 키워드 포함
- 브랜드명은 끝에 배치

### Description (설명)
- 150자 이내
- 행동 유도 문구 포함
- 타겟 키워드 자연스럽게 포함

### Slug (URL)
- 영문 소문자 + 하이픈 사용
- 타겟 키워드 포함
- 간결하게 유지

---

## 4. 이미지 최적화

- Alt 텍스트 필수
- 파일명에 키워드 포함
- WebP 포맷 권장
- 적절한 압축 적용

---

## 배포 전 체크리스트

### 구조
- [ ] H1 태그가 1개인가?
- [ ] TL;DR 섹션이 있는가?
- [ ] FAQ 섹션이 있는가? (최소 3개)
- [ ] 내부 링크 섹션이 있는가? (최소 2개)

### 메타데이터
- [ ] title이 60자 이내인가?
- [ ] description이 150자 이내인가?
- [ ] slug가 적절한가?
- [ ] tags가 설정되어 있는가?

### 콘텐츠
- [ ] 타겟 키워드가 자연스럽게 분포되어 있는가?
- [ ] 이미지에 alt 텍스트가 있는가?
- [ ] Disclaimer가 포함되어 있는가?

---

## 위반 시 조치

내부 링크 섹션이 누락된 경우:
```
[ERROR] 배포 불가: 내부 링크 섹션이 누락되었습니다.
templates/blog-post.mdx의 "관련 글" 섹션을 참고하세요.
```
