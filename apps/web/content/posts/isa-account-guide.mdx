## 핵심 요약

- ISA(개인종합자산관리계좌)는 한 계좌 안에서 예금·펀드·ETF·국내주식을 굴리고, **이익의 일부를 비과세**로 챙길 수 있는 계좌예요
- 일반형은 **200만원**, 서민형·농어민형은 **400만원**까지 비과세. 초과분은 9.9% 분리과세(일반 금융소득세 15.4%보다 낮아요)
- 의무 가입 기간 **3년**, 연 납입 한도 **2,000만원**, 총 한도 **1억원**이에요
- 지금 당장 ISA를 개설하고 납입한 날부터 3년 카운트가 시작돼요

---

## ISA가 중요한 이유

주식이나 펀드로 1,000만원 이익을 냈다고 가정해봐요.

- **일반 계좌**: 154만원 세금 납부 (15.4%)
- **ISA 계좌**: 200만원까지 비과세, 나머지 800만원에 9.9% = 79.2만원 세금

**차이: 74.8만원.** 이게 1년이 아니라 10년 동안 쌓이면 어마어마해요.

ISA는 정부가 만든 절세 도구예요. 사용하지 않는 것은 세금을 자발적으로 더 내는 거나 다름없어요.

<Callout type="info" title="ISA란?">
**ISA (Individual Savings Account)**: 개인종합자산관리계좌.
한 계좌에서 다양한 금융상품을 운용하고, 계좌 내 손익을 통산한 뒤 비과세 혜택을 적용해요. 2016년 도입.
</Callout>

## 섹션 1: ISA 종류 3가지

### 중개형 ISA ← 대부분 이걸 쓰면 돼요

| 항목 | 내용 |
|------|------|
| 운용 방식 | 본인이 직접 투자 (주식, ETF, 펀드, RP 등) |
| 국내주식 투자 | 가능 ✓ |
| 해외주식 직접 투자 | 불가 (해외 ETF는 가능) |
| 가입 조건 | 국내 거주자 (소득 무관) |
| 수수료 | 증권사 수수료 (ETF는 0.01%대) |
| 특징 | 자유도 높음, 직접 종목 선택 |

### 신탁형 ISA

| 항목 | 내용 |
|------|------|
| 운용 방식 | 은행이 정해준 상품 중 선택 |
| 국내주식 투자 | 불가 |
| 가입 조건 | 국내 거주자 |
| 수수료 | 신탁보수 (연 0.1~0.5%) |
| 특징 | 예금, 펀드 위주, 은행 친화적 |

### 일임형 ISA

| 항목 | 내용 |
|------|------|
| 운용 방식 | 증권사·은행이 대신 운용 |
| 국내주식 투자 | 가능 (직접은 아님) |
| 가입 조건 | 국내 거주자 |
| 수수료 | 운용보수 (연 0.5~1.5%) |
| 특징 | 초보자용, 알아서 운용해줌 |

**대부분의 경우 중개형 ISA가 정답이에요.** 직접 ETF를 사고팔 수 있고, 수수료도 가장 낮아요.

## 섹션 2: 비과세 혜택 구조

### 어떻게 세금이 줄어드나요?

ISA는 계좌 안에서 발생한 모든 손익을 **통산(합산)**해요.

**예시:**
- A 펀드 이익: +500만원
- B ETF 손실: -200만원
- **순이익 = 300만원**

일반 계좌라면 A 펀드 이익 500만원에 세금이 부과되지만, ISA에서는 300만원에만 세금이 부과돼요. 게다가 비과세 한도까지 제외할 수 있어요.

### 유형별 비과세 한도

| 구분 | 비과세 한도 | 초과분 세율 | 일반 세율 |
|------|-----------|-----------|---------|
| 일반형 | 200만원 | 9.9% | 15.4% |
| 서민형 | 400만원 | 9.9% | 15.4% |
| 농어민형 | 400만원 | 9.9% | 15.4% |

### 서민형 조건

- 직전 3년 금융소득 합산 과세 대상이 아닌 자
- 소득 기준: 근로소득 5,000만원 이하 또는 종합소득 3,800만원 이하
- 가입 시 소득 확인 서류 제출 필요

<Callout type="info" title="금융소득종합과세자는 ISA 가입 불가">
직전 3년 중 1번이라도 금융소득(이자+배당)이 연 2,000만원을 초과해 금융소득종합과세 대상이 됐다면 ISA에 가입할 수 없어요.
</Callout>

## 섹션 3: 납입 한도와 의무 가입 기간

### 기본 구조

| 항목 | 내용 |
|------|------|
| 연 납입 한도 | 2,000만원 |
| 총 납입 한도 | 1억원 |
| 의무 가입 기간 | 3년 |
| 중도 해지 | 가능 (단, 비과세 혜택 소멸) |
| 만기 연장 | 가능 (1년 단위) |

### 미납입 한도 이월 가능해요

| 연도 | 납입액 | 미납분 | 누적 이월 가능 한도 |
|------|--------|--------|-------------------|
| 1년차 | 1,000만원 | 1,000만원 | - |
| 2년차 | 500만원 | 1,500만원 | 이월: 1,000만원 |
| 3년차 | 최대 3,500만원 납입 가능 | - | (2,000+1,000+500) |

직장인이 한 번에 목돈을 넣기 어려울 때, 이전 연도 미납 한도를 이월해 나중에 한꺼번에 활용할 수 있어요.

## 섹션 4: 계좌 안에서 뭘 살 수 있나요?

### 중개형 ISA 투자 가능 상품

| 상품 | 투자 가능 여부 |
|------|-------------|
| 국내 주식 (코스피·코스닥) | ✅ 가능 |
| 국내 ETF | ✅ 가능 |
| 해외 ETF (미국 상장) | ✅ 가능 |
| 국내 펀드 | ✅ 가능 |
| 예금·RP | ✅ 가능 |
| 해외 주식 직접 투자 | ❌ 불가 |
| 가상자산 | ❌ 불가 |

### 추천 활용 방식

**장기 적립 투자자라면:**
- S&P 500 추종 ETF (TIGER 미국S&P500, ACE 미국S&P500 등)
- 월 50~100만원 정기 매수

**배당 수익 원하는 투자자라면:**
- 국내 고배당 ETF (KODEX 배당가치 등)
- 배당소득세를 ISA 내에서 처리하면 절세 효과

**안정형 투자자라면:**
- 단기채 ETF + 예금 혼합
- ISA 내 이자 소득도 비과세 혜택 적용

<Callout type="warning" title="ISA 내 국내 주식 매매차익은 비과세">
ISA 안에서 국내 주식을 매도하면 매매차익이 비과세예요. 일반 계좌에서는 대주주가 아닌 경우 비과세이지만, ISA에서는 손익 통산과 비과세 한도 적용이 함께 돼요. ETF 매매차익은 일반 계좌에서도 과세 대상이니 ISA 효과가 더 커요.
</Callout>

## 섹션 5: ISA 만기 후 활용법

### 만기 시 선택지

3년 의무 기간이 지나면 크게 세 가지 선택을 할 수 있어요.

**1. 해지 후 현금화**
- 비과세 혜택 적용 후 수령
- 가장 단순한 방법

**2. 연금계좌로 이전 (절세 혜택 추가)**
- ISA 만기 금액을 연금저축·IRP로 이전 시 이전 금액의 **10% (최대 300만원)**를 세액공제
- 연간 최대 900만원 세액공제 한도에서 추가로 300만원 더 공제받는 효과
- 장기 자산 형성에 최적

**3. 계속 유지 (1년 단위 연장)**
- 비과세 혜택은 이미 소멸됐지만 9.9% 분리과세 계속 적용
- 한도를 최대한 활용하려면 연장이 유리할 수 있어요

<Callout type="info" title="ISA → 연금계좌 이전이 최강의 절세 콤보">
ISA 만기 자금을 연금저축이나 IRP로 이전하면:
1. ISA 비과세 혜택 소멸 없이 이전 가능
2. 이전 금액의 10% (최대 300만원) 추가 세액공제
3. 연금으로 받으면 연금소득세 3.3~5.5% 적용 (낮은 세율)

30대에 ISA로 적립 → 55세 이후 연금으로 수령하는 장기 플랜이 가장 효율적이에요.
</Callout>

- [ ] 자동이체 설정 (월 일정 금액 납입)

| 증권사 | 특징 |
|--------|------|

## FAQ

<Accordion title="Q1. ISA는 1인 1계좌인가요?">
네, ISA는 1인 1계좌예요. 여러 금융기관에서 동시에 개설할 수 없어요. 다른 곳에 ISA가 있다면 먼저 해지하거나 이전한 뒤 새로 개설해야 해요.
</Accordion>

<Accordion title="Q2. 월급쟁이도 가입할 수 있나요?">
근로소득자라면 가입 가능해요. 직전 연도 근로소득이 있으면 서민형 조건(5,000만원 이하)도 확인해보세요. 첫 직장 취업자라면 소득 확인 서류를 제출하면 돼요.

소득이 없어도 가입은 가능하지만, 소득이 없는 경우 서민형 혜택(400만원 비과세)을 받으려면 조건을 더 꼼꼼히 확인해야 해요.
</Accordion>

<Accordion title="Q3. 중도 해지하면 어떻게 되나요?">
의무 가입 기간(3년) 이전에 해지하면 비과세 혜택이 소멸해요. 그동안 발생한 이익 전부에 일반 세율(15.4%)이 적용돼요.

단, 특별 중도 해지 사유(천재지변, 사망·해외이민, 3개월 이상 입원 치료 등)에 해당하면 혜택을 유지하면서 해지 가능해요.
</Accordion>

<Accordion title="Q4. ISA와 연금저축을 같이 해야 하나요?">
가능하다면 둘 다 하는 게 유리해요.

- **ISA**: 3년 단위 유연 운용, 비과세 + 낮은 분리과세
- **연금저축**: 연 600만원 한도 세액공제, 노후 자금 형성

두 계좌를 함께 운용하고 ISA 만기 시 연금계좌로 이전하면 세액공제 혜택이 추가로 생겨요. 여력이 된다면 ISA 납입 → 연금저축 세액공제 → ISA 만기 시 연금계좌 이전이 가장 효율적인 3단 콤보예요.
</Accordion>

<Accordion title="Q5. ISA 안에서 손실이 나면 어떻게 되나요?">
ISA는 계좌 전체 손익을 통산해요. A 펀드에서 100만원 이익, B 주식에서 200만원 손실이면 전체 순손실 100만원 → 세금 없어요.

다만 순손실 상태에서 해지하면 비과세 한도는 소멸하고, 손실에 대한 세금 환급은 없어요.
</Accordion>
