## 핵심 요약

- 청약 가점제는 **무주택 기간(32점) + 부양가족 수(35점) + 청약통장 가입 기간(17점)**, 총 **84점 만점**이에요
- 서울 인기 단지 당첨 커트라인은 보통 **60점 이상**, 강남권은 70점대 이상이에요
- 가점이 낮다면 추첨제 비율이 높은 비규제 지역이나 85㎡ 초과 물건을 노리는 게 현실적이에요
- 청약통장은 오래 유지할수록 유리해요. 지금 당장 해지하지 마세요

---

## 왜 청약 가점제가 중요한가

서울 분양 아파트 한 채에 수천 명이 몰려요. 내 집 마련의 기회를 공정하게 배분하기 위해 만들어진 기준이 바로 **가점제**예요.

가점제는 "오래 기다린 실수요자"를 우선시하는 원칙이에요. 하지만 막상 내 점수가 몇 점인지, 어디를 노릴 수 있는지 제대로 아는 사람은 많지 않아요.

지금 내 점수를 정확히 계산하고, 전략을 세울 수 있어요.

<Callout type="info" title="가점제 vs 추첨제">
**가점제**: 항목별 점수 합산, 고득점자 당첨 (주로 투기과열지구 전용면적 85㎡ 이하)
**추첨제**: 무작위 추첨 (가점이 낮아도 도전 가능)

같은 단지도 면적별, 지역별로 가점제·추첨제 비율이 달라요. 입주자 모집공고를 반드시 확인해야 해요.
</Callout>

## 섹션 1: 84점 만점의 구조

청약 가점은 3가지 항목으로 구성돼요.

### 항목 1. 무주택 기간 (최대 32점)

| 무주택 기간 | 점수 |
|-----------|------|
| 1년 미만 | 2점 |
| 1년 이상 ~ 2년 미만 | 4점 |
| 2년 이상 ~ 3년 미만 | 6점 |
| 3년 이상 ~ 4년 미만 | 8점 |
| 4년 이상 ~ 5년 미만 | 10점 |
| 5년 이상 ~ 6년 미만 | 12점 |
| 6년 이상 ~ 7년 미만 | 14점 |
| 7년 이상 ~ 8년 미만 | 16점 |
| 8년 이상 ~ 9년 미만 | 18점 |
| 9년 이상 ~ 10년 미만 | 20점 |
| 10년 이상 ~ 11년 미만 | 22점 |
| 11년 이상 ~ 12년 미만 | 24점 |
| 12년 이상 ~ 13년 미만 | 26점 |
| 13년 이상 ~ 14년 미만 | 28점 |
| 14년 이상 ~ 15년 미만 | 30점 |
| 15년 이상 | 32점 |

<Callout type="warning" title="무주택 기간 산정 주의사항">
- **만 30세 이후**부터 산정해요 (30세 이전은 무주택이어도 인정 안 됨)
- 단, 만 30세 이전 결혼했다면 결혼한 날부터 산정해요
- 세대원 중 한 명이라도 주택을 소유했다면 해당 기간은 무주택 기간에서 제외돼요
- 입주자 모집공고일 기준으로 산정해요
</Callout>

### 항목 2. 부양가족 수 (최대 35점)

| 부양가족 수 | 점수 |
|-----------|------|
| 0명 | 5점 |
| 1명 | 10점 |
| 2명 | 15점 |
| 3명 | 20점 |
| 4명 | 25점 |
| 5명 | 30점 |
| 6명 이상 | 35점 |

<Callout type="warning" title="부양가족 인정 기준">
- **배우자**: 무조건 포함
- **직계존속(부모·조부모)**: 청약자 또는 배우자의 주민등록상 동거 + 3년 이상 동일 주소 등재 필요
- **직계비속(자녀)**: 만 30세 미만 미혼 자녀, 또는 자녀가 결혼했다면 별도 세대 구성 시 제외
- 형제·자매는 인정 안 돼요

부양가족 등록은 실제 거주 여부가 핵심이에요. 서류만 맞춰놓으면 당첨 취소 + 형사처벌 대상이에요.
</Callout>

## 섹션 2: 내 점수 계산 예시

### 사례 A: 35세, 결혼 3년차, 자녀 1명

| 항목 | 상황 | 점수 |
|------|------|------|
| 무주택 기간 | 만 30세부터 5년 무주택 | 12점 |
| 부양가족 | 배우자(1) + 자녀(1) = 2명 | 15점 |
| 청약통장 | 13년 가입 | 15점 |
| **합계** | | **42점** |

→ 서울 일반 지역 도전 가능. 강남·마포는 어려울 수 있어요.

### 사례 B: 45세, 부모님과 동거, 자녀 2명

| 항목 | 상황 | 점수 |
|------|------|------|
| 무주택 기간 | 15년 이상 | 32점 |
| 부양가족 | 배우자(1) + 부모(2) + 자녀(2) = 5명 | 30점 |
| 청약통장 | 15년 이상 | 17점 |
| **합계** | | **79점** |

→ 서울 어디든 최상위권으로 도전 가능해요.

### 사례 C: 28세, 미혼

| 항목 | 상황 | 점수 |
|------|------|------|
| 무주택 기간 | 30세 미만 → 0년 인정 | 2점 (최소) |
| 부양가족 | 0명 | 5점 |
| 청약통장 | 5년 가입 | 7점 |
| **합계** | | **14점** |

→ 가점제 사실상 불가. 추첨제 물량이나 지방 청약을 노려야 해요.

## 섹션 3: 실제 당첨 커트라인

### 서울 지역 최근 당첨 가점 현황 (참고용)

| 지역 | 일반적 커트라인 | 최고 커트라인 |
|------|--------------|-------------|
| 강남·서초·송파 | 70점 이상 | 84점 (만점) |
| 마포·용산·성동 | 65점 이상 | 75점 이상 |
| 노원·도봉·강북 | 55점 이상 | 70점 이상 |
| 경기 과천·성남 | 60점 이상 | 75점 이상 |
| 경기 일반 | 40~55점 | 65점 이상 |
| 지방 광역시 | 30~45점 | 60점 이상 |

*출처: 한국부동산원 청약HOME 공개 당첨자 가점 데이터, 2024~2025년 기준*

<Callout type="danger" title="커트라인은 단지마다 달라요">
위 수치는 참고용이에요. 같은 노원구라도 브랜드 아파트와 소규모 단지의 커트라인은 완전히 달라요. 청약 전에 반드시 **청약HOME**에서 해당 단지 유사 공고 당첨 가점을 확인하세요.
</Callout>

## 섹션 4: 내 점수대별 전략

### 50점 이상: 서울 비강남권 도전

- 서울 일반 지역 가점제 물량 중 경쟁이 덜한 단지 노리기
- 85㎡ 이하 전용 가점 물량 집중 공략
- 인기 없는 중간층, 북향 세대는 커트라인이 낮을 수 있어요

### 40~50점: 경기 2기 신도시 또는 서울 추첨제

- 서울 85㎡ 초과 추첨제 물량 (가점 무관)
- 경기 일반 지역 가점제 도전 가능
- 신혼희망타운, 생애최초 특별공급 별도 검토

### 30~40점: 지방 또는 특별공급

- 지방 광역시 가점제 도전
- **신혼부부 특별공급**: 결혼 7년 이내, 소득 기준 충족 시 유리
- **생애최초 특별공급**: 생애 첫 주택 구매, 소득·자산 기준 있음

### 30점 미만: 추첨제 집중

- 비규제 지역 추첨제 100% 물량
- 85㎡ 초과 추첨제 물량
- 청약통장 꾸준히 유지하며 점수 쌓기

<Callout type="info" title="특별공급도 반드시 확인하세요">
신혼부부, 생애최초, 다자녀, 노부모 부양, 기관추천 등 특별공급은 일반공급과 별도로 공급돼요. 조건이 맞는다면 특별공급이 당첨 확률이 훨씬 높을 수 있어요.
</Callout>

## FAQ

<Accordion title="Q1. 청약통장을 해지하면 점수가 어떻게 되나요?">
해지하면 가입 기간이 0으로 초기화돼요. 다시 가입해도 기존 기간은 인정되지 않아요. 점수가 낮아서 당분간 청약을 안 할 것 같아도, 통장은 절대 해지하지 마세요.

청약통장 유지 비용은 월 2만원 이상 납입이면 충분하고, 10년 이상 유지하면 금리 우대 혜택도 있어요.
</Accordion>

<Accordion title="Q2. 결혼하면 가점이 유리해지나요?">
일반적으로 유리해요.

- **배우자 포함**: 부양가족 +1명 (10점)
- **무주택 기간**: 결혼 전 30세 미만이었다면, 결혼일부터 무주택 기간 산정 가능
- **통장 합산**: 부부가 각자 통장을 보유하면 둘 중 오래된 것 기준 사용

단, 배우자가 과거 주택을 소유했다면 그 기간은 무주택 기간에서 제외될 수 있어요.
</Accordion>

<Accordion title="Q3. 분양가 상한제 지역은 무조건 가점제인가요?">
분양가 상한제 지역(투기과열지구 포함)에서 전용 85㎡ 이하는 **100% 가점제**예요.

85㎡ 초과는 50% 가점제 + 50% 추첨제로 나뉘어요.

규제 지역이 아닌 경우에는 지역·단지별로 가점제 비율이 달라요. 입주자 모집공고를 반드시 확인하세요.
</Accordion>

<Accordion title="Q4. 부모님을 부양가족에 넣으려면 얼마나 준비해야 하나요?">
부모님을 부양가족으로 인정받으려면:

1. **주민등록 동거**: 청약 신청일 기준 같은 주소에 등록
2. **3년 이상 동일 등재**: 주민등록상 3년 이상 함께 등재되어 있어야 해요
3. **60세 이상**: 만 60세 이상 직계존속 (일부 예외 있음)
4. **무주택**: 부모님도 무주택이어야 해요

단기간에 맞추려다 행정처분을 받을 수 있어요. 장기적으로 실제 거주 후 자연스럽게 준비하는 것이 안전해요.
</Accordion>

<Accordion title="Q5. 점수가 같으면 어떻게 결정되나요?">
동점자가 나올 경우 다음 순서로 결정돼요:

1. 무주택 기간이 긴 사람
2. 부양가족 수가 많은 사람
3. 청약통장 가입 기간이 긴 사람
4. 그래도 동점이면 추첨으로 결정해요
</Accordion>
